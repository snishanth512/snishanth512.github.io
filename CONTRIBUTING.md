# Contribution Guidelines

Thank you for your interest in improving my portfolio!

## How to Contribute

1. **Report Issues:** If you find a typo or a broken link, please open a GitHub issue.
2. **Pull Requests:** 
   - Fork the repository.
   - Create a new branch for your changes.
   - Ensure your Markdown is well-formatted.
   - Submit a pull request with a clear description of the change.

I aim to review all contributions within a week.
